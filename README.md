> A plugin for [Voiden](https://github.com/VoidenHQ) — the developer-first API client.

# Simple Assertions

Add response assertions to your requests using an easy-to-fill table. No scripting required.

## Features

- Two-column assertion table (expression / expected value)
- Support for JSONPath, XPath, and form field assertions
- Multiple assertion operators: equals, contains, exists, matches, and more
- Assertion results displayed as a sub-panel in the response view
- Color-coded pass / fail indicators

## Usage

Use the `/assertions` slash command to insert an assertions table below your request block. Add rows with the expression to check on the left and the expected value on the right.
